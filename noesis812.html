📚 Library — Button Reference
Adding & Opening Books
ADD BOOKS Import an EPUB file from your device. The book is stored locally in IndexedDB — nothing is uploaded anywhere
Book cover / title Click the cover thumbnail or book title to open it in the Reader, restoring the last saved position and settings
Chapters · Snapshots badges Two badges on each book card show at a glance how many chapters have been extracted and how many total snapshots are saved for that book
Delete book Permanently removes the book and all its extracted chapters and snapshots from the library
Extracted Chapters
Chapter name Click to reopen the chapter in Noesis Editor with the most recent snapshot loaded
Snapshot count badge Shows the number of saved versions for that chapter (e.g. "3 snapshots"). Appears next to the chapter name
Delete chapter Removes the chapter and all its snapshots. The original book in the library is not affected
Snapshots
Snapshot row Each row is a saved version of a chapter. Click to reopen that exact version in Noesis Editor — text, highlights and annotations included. The description label (e.g. "clean", "annot", or a custom name) and the save date are shown on each row
Green dot Marks the most recent snapshot for that chapter
Delete snapshot Removes only that version — other snapshots for the same chapter are preserved
Interface
THEMES Switches between light (editorial) and dark library themes. Preference is saved in localStorage
IMPORT SNAPSHOTS Reimport .html snapshot files previously saved by Noesis Editor to disk. On desktop: select files from your Downloads folder. On Android: pick the files manually. Duplicates are automatically skipped
EDITOR Open Noesis Editor in standalone mode (blank document). Use this for free writing or chunk assembly without loading a specific chapter
TOOLS Opens a dropdown menu with three external web tools: noesis-epub-tools — web app for editing and managing EPUB files; Pandoc Online — universal document format converter (Markdown, DOCX, HTML, LaTeX, EPUB, ODT and more), runs entirely in the browser via pandoc.wasm without uploading files; Mozilla PDF Viewer — PDF.js browser-based PDF reader with full accessibility support
