Features

Everything you need to really read

Noesis combines reading, translation, annotation, versioning and document creation in a single HTML file. No installation, no account.

🌍

Multilingual reading

Instant translation via native browser. Zero external APIs, zero per-character costs. Read foreign bestsellers on release day.

🎨

3-colour Highlighting

Yellow, green, pink with selective removal. Highlights are preserved in annotated snapshots.

🗂️

Hierarchical chapter extraction

Extract the current chapter or the entire branch (chapter + all sublevels). Inline base64 images, full toolbar ready.

v810
📸

Snapshot System

Save successive versions of the extracted chapter directly to the filesystem. Reload any version with one click. Real, portable HTML files.

v810
🖊️

Excalidraw integrated

Create diagrams, flowcharts and mind maps directly in the editor. Export as SVG or PNG. Saved locally in the browser.

📦

Chunk collection

Select text, images and tables → added to the collection. Exportable as JSON, importable into the editor to build documents.

✏️

Integrated WYSIWYG Editor

Full formatting, tables with floating toolbar, inline base64 images. Export to PDF, DOCX or HTML for CMS.

🔒

Zero server, zero tracking

Everything local. Books in EpubLibraryDB, snapshots in noesisDB, preferences in localStorage. Not a byte leaves the device.

🛠️

Helper tools included

ePub Slimmer, Multi ePub Slimmer and ePub Splitter: compress, batch-process and extract chapters — all in the browser.

Privacy

Zero server. Zero tracking.

Fully local architecture — not a byte of your books, annotations or snapshots leaves the device.

  • ✓ Books saved in IndexedDB (EpubLibraryDB) — local
  • ✓ Chapter metadata in IndexedDB (noesisDB) — local, snapshots on filesystem
  • ✓ Preferences in localStorage — local
  • ✓ No communication with external servers
  • ✓ Works offline after first load
  • ✓ No account, no subscription
  • ✓ Automatic GDPR compliance (privacy by design)
