Additional tools

Four tools to manage your EPUBs: high-quality PDF Viewer, single and batch image compression, and splitter to extract chapters as separate files. All in the browser.

Tool 1

Mozilla PDF.js Viewer

High-quality rendering of extracted chapters via Mozilla's open-source PDF viewer. Integrated into the chapter export flow.

📖

PDF.js Viewer

High-quality PDF rendering directly in the browser

Browser native Mozilla

📖 Advanced viewing

High-quality PDF rendering with support for embedded fonts, complex layouts and accurate colours — superior to the browser print dialog.

🔍 Zoom and navigation

Zoom in and out with precise percentage, page navigation with direct number input, full-screen view.

🔎 Text search

Full-text search in the PDF with highlighted results. Useful for quickly finding specific terms in long chapters.

🖨️ Direct printing

Print the PDF directly from the viewer using browser native functionality, with print-optimised layout.

📖 PDF.js Viewer — high-quality rendering of the extracted chapter
ℹ️
How to access the PDF Viewer — The PDF Viewer is activated from the 👁 PDF Viewer button in the extracted chapter toolbar. It is not a separate page — it is integrated into the chapter export flow.
Tool 2

ePub Slimmer

Compresses images in an EPUB file to reduce size. Useful for EPUBs with many high-resolution images that slow down loading.

🗜️

ePub Slimmer

Image compression for a single EPUB file

1 file JPG · PNG · WebP
Open tool →

🖼️ Image compression

Compresses all images in the EPUB to JPEG with configurable quality. Maintains aspect ratio and image structure.

⚙️ Configurable quality

Adjust compression quality (0–100). Values between 60–80 offer a good balance between size and quality for most EPUBs.

🔒 No DRM

Works only with DRM-free EPUBs. Protected files cannot be processed.

Tool 3

Multi ePub Slimmer

Batch version of ePub Slimmer. Process multiple EPUB files simultaneously with the same compression settings.

📦

Multi ePub Slimmer

Batch compression of up to 10 EPUB files

Up to 10 files Parallel batch
Open tool →

🔢 Queue with individual status

Each file shows its own status: pending → processing → done or error.

📚 Batch processing

Select multiple EPUBs at once. Each one is compressed independently and downloaded automatically.

📊 Progress

Progress bar for each EPUB during processing. Processed files appear in the results list as they complete.

⚠️ Error handling

If one file fails (corrupt EPUB, unsupported format) the others continue processing. Clear error messages for each problematic file.

Tool 4

ePub Splitter

Splits an EPUB into individual chapter files. Each chapter is saved as a separate EPUB — useful for reading tools that handle large EPUBs poorly.

✂️

ePub Splitter

Extract selected chapters as separate EPUB files

TOC-based Multi-select
Open tool →

✂️ Split by chapter

Each item in the EPUB spine becomes a separate EPUB with correct metadata. Chapter names are taken from the TOC.

📦 ZIP download

All split EPUBs are packaged in a single ZIP file for download. No intermediate steps.

ℹ️
Difference from Extract Chapter — The Splitter produces standalone .epub files (importable in other readers). The Extract Chapter function in Noesis produces standalone HTML documents with the integrated Noesis toolbar. They are complementary tools with different purposes.
