Noesis Editor
✏️ Noesis Editor — Button Reference
Chapter
New Clear the editor and start a blank document. If the editor has content, a confirmation dialog asks whether to save first
Import Load a chapter into the editor. When opened from the Library, shows the list of saved snapshots from IndexedDB. Also lets you pick any .html file from disk
Save Export the chapter as two simultaneous .html files — one clean (background colors removed) and one annotated (as-is). Both include Noesis metadata tags for re-import into the Library
More formats Additional chapter export options: TXT (plain text), Markdown (.md), MD + images ZIP (images extracted to a folder), JSON doc (raw HTML wrapped in JSON), Print / PDF (browser print dialog), DOCX
Collection
+ N (Add) Add the current selection to the collection. Select a paragraph (triple-click), a table cell, or an image before clicking. The counter shows how many chunks are stored
Import JSON Load a previously exported .json collection file back into the collection
Export JSON Save the full collection as a structured .json file (version, chapter name, all chunks with type and content)
More formats Additional collection export options: Markdown (text only, tables as raw HTML, images omitted), MD + images ZIP (images extracted), HTML collection (standalone HTML page with all chunks)
Inspect Open the collection panel — review chunks, select/deselect, inject selected chunks back into the editor, delete individual chunks, or view any chunk fullscreen
Clear Remove all chunks from the collection after confirmation. If the collection is already empty, shows a toast instead
Inspect Panel
Fullscreen View the content of a single chunk in a fullscreen overlay. Shows the chunk label (number and type)
Select All / None Select or deselect all chunks in the panel. The counter shows how many are selected
Inject into Editor Insert all selected chunks into the editor at the current cursor position. Enabled only when at least one chunk is selected
Tools
Excalidraw Opens Noesis Excalidraw in a new tab — a fork of Excalidraw for creating diagrams, flowcharts, mind maps and visual sketches. Drawings are saved locally in the browser and can be exported as SVG or PNG
